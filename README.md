# Item Management Module — SliitArena 360

## What's included
- **Equipment**: CRUD for sports equipment with stock tracking (`/api/equipment`)
- **Reservations**: Students reserve/return items, admin manages returns, low-stock alerts
- **Frontend Pages**: Login, Register, Dashboard, EquipmentList (with reserve/return UI)

## Setup
```bash
cd server && npm install && npm run dev   # port 5003
cd client && npm install && npm start
```
