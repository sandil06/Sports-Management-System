# Event & Payment Management Module — SliitArena 360

## What's included
- **Events**: CRUD for matches/training/tournaments (`/api/events`)
- **Tournaments**: Full tournament lifecycle with individual & team registration (`/api/tournaments`)
- **Sponsors**: Sponsor registration, login, apply to sponsor tournaments, admin approval, contribution tracking (`/api/sponsors`)
- **Frontend Pages**: Events, EventForm, TournamentList, TournamentForm, Sponsors, SponsorDashboard, SponsorApprovals, MySponsorships

## Setup
```bash
cd server && npm install && npm run dev   # port 5004
cd client && npm install && npm start
```
