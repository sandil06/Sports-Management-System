# Facility Management Module — SliitArena 360

## What's included
- **Facilities**: CRUD for courts/fields/gyms/pools (`/api/facilities`)
- **Bookings**: Book a facility, check availability, cancel booking, admin view all bookings
- **Frontend Pages**: Login, Register, Dashboard, FacilityList (with booking UI)

## Setup
```bash
cd server && npm install && npm run dev   # port 5002
cd client && npm install && npm start
```
